# Grill Me

Interview the user relentlessly about every aspect of this plan until we reach a shared understanding. Walk down each branch of the design tree, resolving dependencies between decisions one-by-one. For each question, provide your recommended answer.

## Rules

1. Ask questions **one at a time** — do not dump a list
2. If a question can be answered by exploring the codebase, explore the codebase instead of asking
3. After each answer, decide: drill deeper into that branch, or move to the next branch
4. When all branches are resolved, produce a summary of decisions made
